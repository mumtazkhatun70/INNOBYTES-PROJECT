Personal Finance Management Application

Overview

This application allows users to manage their personal finances by registering, logging in, adding, viewing, and deleting transactions, generating monthly financial reports, and backing up/restoring the database.

Features

  • User registration and authentication
  • Add, view, and delete income and expense transactions
  • Generate monthly financial reports
  • Backup and restore database data

Setup Instructions

  1. Clone the repository: 
    git clone https://github.com/yourusername/finance-app.git
                    cd finance-app
  2. Install dependencies: 
    pip install -r requirements.txt
  3. Initialize the database: 
    python init_db.py
  4. Run the application: 
    python main.py

Usage Guide

  • Register a new user
  • Login using your credentials
  • Add transactions by specifying type, category, amount, date, and description
  • View all transactions for your account
  • Generate monthly reports by selecting month and year
  • Backup your database to a file and restore when needed

Function Overview

Authentication (auth.py)

  • register_user(username, password): Register a new user
  • login_user(username, password): Login and return user ID

Transactions (transactions.py)

  • add_transaction(user_id, type, category, amount, description, date=None): Add income/expense
  • view_transactions(user_id): View all transactions of the user
  • delete_transaction(user_id, transaction_id): Delete a specific transaction

Reports (reports.py)

  • generate_report(user_id, month, year): Generate monthly financial summary

Backup (backup.py)

  • backup_database(): Backup database to a file
  • restore_database(): Restore database from backup file

Testing Instructions

Run unit tests to verify the application functionality:

python -m unittest discover tests

The tests cover:

  • User registration and login
  • Transaction add/view/delete
  • Report generation
  • Database backup and restore
